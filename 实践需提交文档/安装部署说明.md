## 前端

### 安装环境

nodejs10.7.0

### 安装步骤

```
cd fronted
```

```
npm install
```

Compiles and hot-reloads for development
```
npm run serve
```

Compiles and minifies for production
```
npm run build
```

### 安装成功测试方法

前端开发时访问浏览器localhost:8080

build打包文件在/dist下

### 常见问题解决

serve运行默认端口是8080，若8080端口被占用，会采用其他端口，在控制台会显示
